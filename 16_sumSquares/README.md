# Exercise 16 - sumSquares

Write a function that sums squares of numbers in an arbitrarily nested array

```javascript
SumSquares([1,2,3]); // 1 + 4 + 9 = 14
SumSquares([[1,2],3]); // 1 + 4 + 9 = 14 
SumSquares([[[[[[[[[1]]]]]]]]]); // 1 = 1
SumSquares([10,[[10],10],[10]]); // 100 + 100 + 100 + 100 = 400
```
